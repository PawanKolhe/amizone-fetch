# Amizone Fetch
REST API to fetch data from Amizone using puppeteer

## Tech Used
* [Node.js](https://nodejs.org/en/)
* [Express](https://expressjs.com/)
* [Puppeteer](https://github.com/puppeteer/puppeteer)

## Environment Setup
### Install dependencies
```bash
npm install
```
### Run locally
```bash
npm run dev
```
